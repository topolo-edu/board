🗂️ sessionStorage 글쓰기

sessionStorage에 저장된 JWT 토큰으로 인증하여 글을 작성합니다

최대 200자까지 입력 가능합니다.

최대 5,000자까지 입력 가능합니다. 내용은 sessionStorage에 자동 저장됩니다.

🔄 자동 토큰 갱신 체험

sessionStorage 방식의 토큰 갱신 과정:

  1. sessionStorage에서 Access Token을 가져와 Authorization 헤더에 포함
  2. API 호출 시 401 에러 발생하면 자동으로 Refresh Token 사용
  3. 새로운 토큰을 sessionStorage에 저장
  4. 실패한 요청을 새 토큰으로 재시도

💡 "토큰 만료 테스트" 버튼을 눌러보세요!

🗂️ sessionStorage 방식 특징

구현 방식

  • • sessionStorage.setItem('accessToken', token)
  • • sessionStorage.getItem('accessToken')
  • • Axios 인터셉터로 자동 헤더 추가

보안 고려사항

  • • JavaScript에서 토큰 접근 가능
  • • XSS 공격 시 토큰 탈취 위험
  • • 브라우저 개발자 도구로 확인 가능
  • • 탭 닫으면 자동으로 토큰 삭제

⏰ sessionStorage 탭 관리 체험

sessionStorage의 탭별 독립성을 체험해보세요:

  1. 이 페이지에서 로그인하고 글을 작성해보세요
  2. 새 탭에서 같은 페이지를 열어보세요 (로그인 안됨)
  3. 원래 탭에서는 여전히 로그인 상태인지 확인해보세요
  4. 탭을 닫고 다시 접속하면 로그아웃됩니다

💡 localStorage와 비교해보세요!
